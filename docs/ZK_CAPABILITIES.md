```bash
# Deploy contract with ZK privacy features
./build/lightchain-cli dev deploy PrivateToken.sol --zk-privacy

```

### **2. Create ZK Rollup**
```bash
# Launch ZK rollup for your application

  --name "GameFi Rollup" \
  --tps 50000 \
  --proof-system starks

```

### **3. Test Privacy Features**
